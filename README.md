# Kali-Linux-Repair-Toolkit
**A simple tool to repair some of the kali linux problems**

# Requirements
- Python3.x 
- Kali Linux
- 
# Installation 
- Open terminal and write this: 
```
git clone https://github.com/srb1mal/kali_linux.git
```
- Atfer download write : 
```
cd kali_linux
```
- And : 
```
python3 kali-repair.py
```
